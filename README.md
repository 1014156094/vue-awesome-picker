## DEMO
![]('https://fyerl.github.io/vue-awesome-picker/static/img/qr-code.png')

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
